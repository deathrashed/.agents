# Getting Started

Claude Skills works with every major AI coding assistant. This section covers installation, your first skill in five minutes, and platform-specific configuration.

## Overview

1. **[Installation](installation.md)** -- Clone the repo, use the skill installer, or copy individual skills
2. **[Quick Start](quick-start.md)** -- Use your first skill in under 5 minutes
3. **[Platforms](platforms.md)** -- Setup guides for Claude Code, Cursor, Copilot, Codex, Windsurf, Cline, and more

!!! tip "Fastest path"
    If you just want to try one skill, copy any `SKILL.md` file directly into your AI assistant's context. No installation required.
