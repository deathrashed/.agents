# marketing-skills

6 flagship marketing skills covering the top of the funnel, mid-funnel conversion and distribution.

## Install (Claude Code)

```bash
/plugin marketplace add borghei/Claude-Skills
/plugin install marketing-skills@claude-skills
```

## Included skills

| Skill | What it does |
|---|---|
| content-creator | Long-form content generation with editorial structure, hooks, SEO bones |
| seo-audit | Technical and content SEO audit, gap analysis, prioritized fix list |
| marketing-demand-acquisition | Demand gen strategy, channel mix, ICP-led campaign design |
| campaign-analytics | Campaign measurement, attribution, funnel analysis, iteration loop |
| landing-page-generator | Landing page copy + structure tuned for a specific offer and audience |
| x-twitter-growth | X/Twitter growth patterns, thread structure, engagement optimization |

## Who this is for

Marketing teams using Claude to scale content, measure what actually converts and grow distribution. Focuses on shipping work, not strategy decks.
