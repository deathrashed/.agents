# compliance-skills

9 flagship compliance skills covering the frameworks regulated industries hit most often.

## Install (Claude Code)

```bash
/plugin marketplace add borghei/Claude-Skills
/plugin install compliance-skills@claude-skills
```

## Included skills

| Skill | Framework |
|---|---|
| information-security-manager-iso27001 | ISO/IEC 27001 ISMS implementation, audit prep, SoA |
| soc2-compliance-expert | SOC 2 Type I/II readiness, control mapping, evidence collection |
| gdpr-dsgvo-expert | GDPR/DSGVO data protection, DPIA, DPO support, breach response |
| eu-ai-act-specialist | EU AI Act risk classification, conformity assessment, technical documentation |
| nis2-directive-specialist | NIS2 cybersecurity governance, incident reporting, supply chain risk |
| dora-compliance-expert | DORA ICT risk management for financial entities, third-party oversight |
| quality-manager-qms-iso13485 | Medical device QMS under ISO 13485 |
| mdr-745-specialist | EU MDR 2017/745 for medical devices, clinical evaluation, technical file |
| fda-consultant-specialist | FDA submissions (510(k), De Novo, PMA), 21 CFR Part 820 QSR |

## Who this is for

Teams in medtech, fintech, AI, healthcare or any regulated sector using AI to accelerate compliance work. The skills enforce framework-accurate terminology and map work products to specific clauses, articles or controls.
