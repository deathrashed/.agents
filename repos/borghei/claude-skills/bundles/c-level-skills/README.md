# c-level-skills

7 flagship C-suite advisory skills for strategic decision-making, board work and executive communication.

## Install (Claude Code)

```bash
/plugin marketplace add borghei/Claude-Skills
/plugin install c-level-skills@claude-skills
```

## Included skills

| Skill | Role |
|---|---|
| ceo-advisor | CEO strategic thinking partner for company strategy, stakeholder management |
| cto-advisor | CTO technical strategy, architecture decisions, build vs. buy, hiring |
| cfo-advisor | CFO capital strategy, unit economics, fundraising, forecasting |
| ciso-advisor | CISO security program design, risk appetite, incident governance |
| cmo-advisor | CMO marketing strategy, positioning, GTM, brand architecture |
| board-meeting | Board meeting prep, agenda design, minute-taking, resolution drafting |
| board-deck-builder | Board deck structure, metric selection, narrative arc, pitch patterns |

## Who this is for

Founders, operators and executives using Claude as a strategic thinking partner. Each skill applies a specific executive lens (finance, tech, security, marketing) plus board-level communication patterns.
