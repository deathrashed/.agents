# product-skills

6 flagship product skills covering strategy, management, delivery and design.

## Install (Claude Code)

```bash
/plugin marketplace add borghei/Claude-Skills
/plugin install product-skills@claude-skills
```

## Included skills

| Skill | What it does |
|---|---|
| product-manager-toolkit | Full PM toolkit: discovery, prioritization, roadmapping, stakeholder mgmt |
| product-strategist | Product strategy, positioning, North Star metric selection |
| agile-product-owner | PO workflows, backlog management, acceptance criteria, sprint reviews |
| product-designer | Product design patterns, information architecture, flow design |
| ux-researcher-designer | UX research methods, interview scripts, synthesis, journey mapping |
| design-system-lead | Design system governance, token architecture, component API design |

## Who this is for

Product teams using Claude to move from discovery through delivery. Covers the PM-PO-designer triangle with framework-accurate methods.
