## How to Download

To download these images for local development:

1. Visit the download URL for each image
2. Convert to WebP format using:
   ```bash
   convert image.jpg -quality 85 -resize 1920x1080 output.webp
   ```
3. Verify file size (<500KB for hero, <200KB for features)
4. Place in appropriate directory under `public/images/`
