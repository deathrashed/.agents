# Map Command

Build a site, topic, entity, editorial, or internal-link architecture map.

## Route

- content-gap-analysis
- internal-linking-optimizer

## Rules

- Turn known opportunities into content architecture, topic map, entity map, and internal-link plan.
- Return clusters, pillar/supporting pages, orphan risks, anchor guidance, and next briefs.

## Output

Return inline artifacts by default. Files may be written only when the user explicitly asks and the runtime can write.
